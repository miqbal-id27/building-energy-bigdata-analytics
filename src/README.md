# Source Code Folder

The portfolio keeps the original notebook implementation under `notebooks/`.

Use this folder later if you want to refactor notebook logic into reusable Python modules, for example:

- `producer.py` for Kafka weather simulation
- `stream_predict.py` for Spark Structured Streaming prediction
- `visualization_consumer.py` for Kafka consumer visualization
- `features.py` for reusable feature engineering logic
